# Aplikasi Perpustakaan Digital
Aplikasi berbasis web untuk pengelolaan buku, anggota, dan peminjaman di perpustakaan

## Fitur
- CRUD Buku
- Pencarian & Pagination
- Manajemen Anggota & Peminjaman
- Login Session untuk Admin & Anggota

# Teknologi
* Codeigniter 3
* Bootstrap + AdminLTE
* MySQL
